This is a full stack ecommerce about furniture, it is fully responsive and contain all the features an ecommerce website have: authentication, shopping cart, product details Page, checkout functionality and a dashboard to manage all products like adding, deleting, updating and filtering products. every product can have multiple images, all the images are stored in the cloud to optimize the performance
